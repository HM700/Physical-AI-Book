# Module 1: ROS 2 Fundamentals

Welcome to the ROS 2 Fundamentals module. This module covers the basics of the Robot Operating System (ROS 2) and its application in robotics development.

## Overview

In this module, you will learn:
- Introduction to ROS 2 architecture
- Nodes, topics, and services
- Working with ROS 2 packages
- Building and running ROS 2 applications
